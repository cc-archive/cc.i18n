Comment ça marche : tout utilisateur qui cliquera sur l'îcone Creative Commons obtiendra votre "Résumé Explicatif" — une source pratique pour faciliter la compréhension de la version complète du contrat. A partir de ce Résumé se trouve un lien vers le "Code Juridique" — le contrat selon les termes duquel vous avez offert votre création. Et à l'intérieur du logo se trouve le "Code Machine" — les métadonnées — qui permet de rechercher automatiquement votre création. Ce processus est illustré dans nos bandes dessinées.
